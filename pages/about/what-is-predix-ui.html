Introduction

What is Predix UI?

Predix UI makes it easy to create powerful, user-friendly web applications with components that just work. The system is made up of three parts:

  • plug-and-play components that speed UI/UX development
  • composable and reusable CSS modules that establish consistent design
  • a quick-start project seed with everything you need to start building now

Used together, these tools help teams jumpstart the creation of Industrial Internet web applications running on top of Predix services and data.

Predix UI Components: UI pieces you need. Already built.

Predix UI Components are a set of user interface pieces that can be quickly snapped together to build an application. Components give you the basic building blocks of a web app, freeing up your time to spend on more important development tasks.

The components are framework agnostic â€” they can fit into many existing JavaScript web applications (e.g. Angular, React, Ember), or serve as a starting point for a new project.

Use the components to do everything from creating a UI foundation for your application to shipping interactive charts that feed off streaming data.

Start using Predix UI Components in your applications. See the Components Quick Start guide.

Predix UI CSS: Consistent, modular design from the start.

Predix UI CSS is a collection of CSS modules that offer consistent design, useful defaults, and customizable styles for web applications. CSS modules can be mixed together to create a strong base for web applications of any size.

Predix UI CSS modules are written in Sass using the powerful BEM (Block-Element-Modifier) and OOCSS (Object-Oriented CSS) code patterns, ensuring your CSS codebase can grow quickly and still be easy to maintain.

Use CSS modules to create the base typography, layout, and content design for your project, or use them to design new custom components unique to your application. Building with Predix UI CSS ensures consistent design across your entire application.

Use the Predix UI CSS Starter Kit to create a consistent design base for your app.

Predix UI Seed: App starter Kit that's ready for production.

Predix UI Seed is a web application starter kit that can be copied, extended and deployed to production immediately. Everything you need is included:

  • A project structure to help teams write organized code using Polymer as an application framework
  • Example Predix UI Components and CSS modules composed to create a dashboard UI
  • Developer tools that speed up the type, save, and reload workflow
  • A Node/Express web server ready to connect to Predix services and APIs

Use the Predix UI Seed to jumpstart your next web application project and create a stable base that your team can extend and build on into the future.
